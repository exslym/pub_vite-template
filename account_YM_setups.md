## NoBounce

Название параметра:

```
NoBounce_5sec
NoBounce_10sec
NoBounce_15sec
NoBounce_30sec
NoBounce_45sec
NoBounce_1min
NoBounce_2min
NoBounce_5min
```

Тип условия: JavaScript-событие

```

NoBounce_10sec
```
